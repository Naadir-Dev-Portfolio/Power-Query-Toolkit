# Power Query — Data Pipeline Templates

This folder contains reusable Power Query (M Language) transformation patterns.

## What's Included

- **Multi-source data merging** — combining data from multiple Excel files/sheets
- **Date table generation** — standard calendar dimension for Power BI models
- **Data cleansing templates** — trimming, type-casting, null handling
- **Unpivot / pivot patterns** — reshaping wide data for analysis

## Usage

Open any `.xlsx` file with Power Query enabled, go to **Data → Get & Transform → Advanced Editor**
and paste the M code. Adjust source references for your environment.

> Note: The full `.xlsm`/`.xlsb` files with embedded Power Query are not included in this
> repo as they contain internal company data. The patterns above are extracted and anonymised.
