# Power-Query-Toolkit

Power Query M templates and patterns for Excel data transformation

Built by [Naadir](https://github.com/Naadir-Dev-Portfolio)

## Overview

Reusable Power Query M language templates for common data transformation tasks. Includes patterns for data cleaning, reshaping, and combining multiple sources. Accelerate your Excel data prep workflows with battle-tested query recipes.

## Features

- M language transformation templates
- Data cleaning patterns (trim, case normalization, etc.)
- Merge and reshape operations
- Multi-source data combination
- Conditional logic and custom columns
- Performance optimization patterns

## Tech Stack

Power Query · M Language · Excel

## Structure

- `templates/` - Ready-to-use Power Query files and examples

## Usage

1. Open templates in Excel's Power Query editor
2. Customize data sources and transformation logic
3. Integrate queries into your workbooks
4. Refresh data on schedule or manually

## Notes

Power Query M syntax reference and best practices included in template files.
